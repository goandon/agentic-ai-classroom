# Episode 6: Give Your Agent a Personality

**System Prompt Design** — 4 building blocks to craft a character agent.

## Files

| File | Description |
|------|-------------|
| `character_chef.py` | Progressive system prompt demo (v0→v3) |

## How to Run

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY="your-key-here"
python character_chef.py
```

## Key Concepts

- **Persona**: Define WHO the AI is
- **Instructions**: Define WHAT it does
- **Guardrails**: Define what it does NOT do
- **Few-shot**: Show HOW it should respond
